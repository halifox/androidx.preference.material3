# Preference-MD3

## 📖 简介
基于 androidx.preference 源码修改的首选项库，使其支持 Material Design 3 (MD3) 风格。

---

## ✨ 功能

- 支持 MD3 风格：修改默认样式，使首选项界面与 Material Design 3 风格完全兼容。

---

## 🎨 设计规范

- [Material Design 3 (MD3)](https://m3.material.io/)
